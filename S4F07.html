Reporting for Finance in SAP S/4HANA (S4F07)

Overview

The course provides an overview and a detailed hands-on to the Embedded Analytics reporting tools in SAP S/4HANA. It describes the main components of the new reporting architecture and shows the necessary tools to set up the different kinds of reports. It also explains the benefits and use case of the different reporting tools.

Goals

This course will prepare you to:

    • Get an understanding of the basic concepts of SAP S/4HANA Analytics
    • Get an introduction to the Virtual Data Model which is the basis for Reporting in SAP S/4HANA
    • Get insight into the SAP S/4HANA Extensibility Tools
    • Have a deeper understanding of the necessary SAP Fiori UI structures for Reporting
    • Use the tools for SAP S/4HANA UIs
