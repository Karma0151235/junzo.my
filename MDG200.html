SAP Master Data Governance: Configuration and Customizing

Overview

In this course, participants will gain understanding and skills related to configuration, as well as the knowledge required to extend and modify the solution. This includes the ability to create a custom solution based on the Custom master data domain. The theoretical lessons are enriched with several hands-on exercises, to deepen the practical knowledge.

Goals

This course will prepare you to:

  • Perform basic configuration of standard MDG content
  • Adjust and extend the standard MDG content
  • Create custom content for MDG

SAP Master Data Governance: Configuration and Customizing

Content

  • MDG Data Quality Management
    • Outline the core processes of data quality management with SAP MDG on SAP S/4HANA
    • Explain master data quality: evaluations, scores, and dimensions
    • Describe how machine learning has been applied in data quality management
  • MDG Data Quality Enhancements
    • Explain search and duplicate checks
    • Explain concepts of BRFplus
    • Explain derivations, validations, and checks
  • MDG Process Modeling
    • Explain workflow basics
    • Explore concepts of process modeling with rule-based workflow
  • MDG Data Replication
    • Consider data export and data import via data transfer
    • Explain concepts of key and value mapping
    • Explain replication usage
    • Explain the configuration setup
  • MDG Custom Objects and MDG Extensibility
    • Explore data modeling concepts
    • Explain data transfer
    • Configure user interfaces
  • MDG Customizing, Setup and Project Strategies
    • Activate and configure the default processes for MDG
    • Explain and use advanced customizing
    • Explain the new and updated functionality that is offered with MDG
    • Start first MDG projects that use the standard data models
