OVERVIEW

This class introduces the participants to develop dynamic, secure, commercial Web sites to reflect the rapidly changing landscape of MySQL and PHP. This will broaden the scope to integrate and implement these technologies by following real-world examples and working sample projects, and also covers related technologies needed to build a dynamic web site.
